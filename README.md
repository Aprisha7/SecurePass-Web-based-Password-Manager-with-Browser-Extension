# SecurePass-Web-based-Password-Manager-with-Browser-Extension
This project is a Web-Based Password Manager with Browser Extension.
It securely stores and manages user credentials using end-to-end encryption and role-based access control (RBAC). The browser extension simplifies password input and retrieval, offering seamless access from web interfaces.

System Architecture
The system follows a multi-tier architecture with modular backend, secure database design, and integrated browser extension workflows.
Core components:
- Frontend (React / HTML-CSS-JS)
- Backend (Node.js / Express.js)
- Database (MongoDB)
- Browser Extension (Manifest v3)
- Encryption Layer (AES-GCM, bcrypt)
